Build a production-ready web application called **TechLab AI** — an enterprise-grade scientific data analysis platform with a modern, intuitive, and high-performance user interface. The application should be designed for researchers, engineers, GIS professionals, GNSS analysts, atmospheric scientists, and students. It must transform raw scientific datasets into cleaned, analyzed, visualized, and exportable results using advanced statistical and AI-powered techniques.

## Core Requirements

Develop a full-stack application using:
- Frontend: React + TypeScript + Next.js (or Vite) + Tailwind CSS + Shadcn/UI + Framer Motion
- Backend: Python FastAPI
- Database: PostgreSQL (SQLite for local mode)
- Cache: Redis
- Background Processing: Celery/RQ with async workers
- Authentication: JWT + OAuth
- Docker support
- REST API + GraphQL
- Modular plugin architecture
